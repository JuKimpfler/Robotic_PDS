# PC Setup for Windows 11

This folder contains everything needed to run the Power Debug Monitor GUI on a Windows 11 laptop.

## Prerequisites
- **Python 3.10+** (install from python.org).
- **Administrator rights** (required for hotspot creation and firewall configuration).
- **Git** (optional, for cloning the repository).
- **Wi‑Fi adapter** that supports hosted network mode.

## Steps
1. **Install Python dependencies**
   ```cmd
   pip install -r ..\requirements.txt
   ```
2. **Create a local Wi‑Fi hotspot**
   The script `setup_windows.bat` uses `netsh` to create a hosted network. It will:
   - Set SSID and key (default: `RoboDebug` / `robodebug123`).
   - Start the hotspot.
3. **Configure Windows Firewall**
   The script automatically adds inbound rules for the UDP telemetry ports used by the system (`5001`, `5002`).
4. **Run the GUI**
   ```cmd
   python ..\rpi5_monitor\main.py
   ```
   The GUI will detect the Windows platform, use the hotspot created above and connect to the Raspberry Pi Zero.

## Customisation
- Edit `setup_windows.bat` to change the SSID, password, or ports.
- If you prefer a different hotspot method, replace the `netsh` commands with your own implementation.
